# SkyMallCore
dotNet Core Demo, Create a Cross Platform WebApp System!</br>
相互学习提升，有不足之处请指教！</br>
有需要急速开发的朋友可以拿来用哦！</br>

1：该项目是由NFine开源项目转化而来 </br>
2：该项目基本框架开发平台是在Asp.net Core 2.0基础上编写</br>
3：旨在促进.Net Core跨平台学习交流，提高开发效率4：.Net Core Mvc + EFCore 等技术，该项目仍会继续完善！</br>

4：根据【基础项目进度】包含：</br>
基本框架结构搭建</br>
Startup 配置注入</br>
加入EFCore</br>
数据处理类封装</br>
依赖注入解决</br>
搭建后台管理</br>
登录</br>
登录首页</br>
登录授权验证</br>
后台首页加载</br>
后台管理员列表</br>
